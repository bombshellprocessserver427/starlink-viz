## 🛠 What is starlink-viz?

starlink-viz shows you the real-time position of Starlink satellites around the Earth. You can view satellites in 3D space or from the ground, see live data from ground stations, and watch how satellites move across the sky. The app uses accurate orbital calculations and lets you track about 10,000 satellites and ground stations. It also shows network paths and performance between satellites.

The interface uses modern web technology but runs as a desktop app on Windows. You do not need coding skills to use it.

## 🛰 Using starlink-viz Features

starlink-viz has several views and tools:

- **Space view:** See satellites orbiting Earth from a space perspective.  
- **Sky view:** View satellite paths as they appear from a ground location.  
- **Live dish telemetry:** Monitor real-time data coming from Starlink dishes connected to the network.  
- **SGP4 propagation:** Watch the predicted movements of satellites using orbital calculations.  
- **ISL routing:** See inter-satellite links and network connections.

You can select specific satellites or ground stations from the sidebar menu to get detailed information.
